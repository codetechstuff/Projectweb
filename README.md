# Projectweb

Simple template for an UrbanClap-like service marketplace.

Files added:
- index.html — main landing page
- css/styles.css — styles
- js/app.js — sample data and interactions

Preview locally:
```bash
# from project root
open index.html            # macOS quick open in browser
# or run a simple static server
python3 -m http.server 8000
# then open http://localhost:8000
```
 
React version
---------------
To run the React (Vite) version:

```bash
npm install
npm run dev
# then open the address printed by Vite (usually http://localhost:5173)
```

Files for the React app live under `src/` and the Vite entry is `index.html`.
